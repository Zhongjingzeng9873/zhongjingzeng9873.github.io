zabbix
Created 2020-04-23| Updated 2020-04-23
Post View:

zabbix是一个基于WEB界面的提供分布式系统监视以及网络监视功能的企业级的开源解决方案。

zabbix能监视各种网络参数，保证服务器系统的安全运营；并提供灵活的通知机制以让系统管理员快速定位/解决存在的各种问题。

zabbix由2部分构成，zabbix server与可选组件zabbix agent。

zabbix server可以通过SNMP，zabbix agent，ping，端口监视等方法提供对远程服务器/网络状态的监视，数据收集等功能，它可以运行在Linux，Solaris，HP-UX，AIX，Free BSD，Open BSD，OS X等平台上。

本次实验是基于LAMP环境搭建zabbix网络监控服务
还有一种是LNMP环境，这里就不多说了。
L: linux
A: apache
N: nginx
M: mysql,mariaDB
P: php,perl,python

1 我们先去关闭selinux

你可以选择临时关闭selinux
#setenforce 0
也可以到 /etc/selinux/config修改配置文件
#vi /etc/selinux/config
SELINUX=disabled
保存退出
1.1临时关闭防火墙
#systemctl stop friwalld
1.2永久关闭防火墙
#systemctl disable firewalld
1.3安装apache
#yum -y install httpd
安装完成后启动apache服务
systemctl enble httpd #开启自启
systemctl start httpd #启动服务
1.4安装mysql
#wget http://dev.mysql.com/get/mysql57-community-release-el7-8.noarch.rpm #这里安装的是mysql5.7版本（需要高版本自行寻找安装包）这个也够用了
安装mysql源
#rpm -ivh mysql57-community-release-el7-8.noarch.rpm
安装mysql
#yum -y install mysql-community-server
设置开机自启
#systemctl enble mysql
启动mysql
#systemctl start mysql
#查看初始密码
#cat /var/log/mysqld.log | grep ‘temporary password’
登录mysql
#mysql -u root -p
输入密码
修改mysql登录密码
ALTER USER ‘root’@’localhost’ IDENTIFIED BY ‘你的密码’;
你想设置简单密码的话需要修改/etc/my.conf
安装mysql源
#rpm -ivh mysql57-community-release-el7-8.noarch.rpm
安装mysql
#yum -y install mysql-community-server
设置开机自启
#systemctl enble mysql
启动mysql
#systemctl start mysql
#查看初始密码
#cat /var/log/mysqld.log | grep ‘temporary password’
登录mysql
#mysql -u root -p
输入密码
修改mysql登录密码
mysqladmin -u root password ykadmin123
ALTER USER ‘root’@’localhost’ IDENTIFIED BY ‘你的密码’;
你想设置简单密码的话需要修改/etc/my.conf
validate_password = off
1.5安装php环境
#yum -y install php php-mysql
1.6安装zabbix
#rpm -ivh http://repo.zabbix.com/zabbix/3.4/rhel/7/x86_64/zabbix-release-3.4-2.el7.noarch.rpm
1.6.1安装zabbix需要的包
#yum install -y zabbix-server-mysql zabbix-get zabbix-web zabbix-web-mysql zabbix-agent zabbix-sender
1.6.2创建一个zabbix库设置为utf-8的字符编码格式（这里需要进入mysql）
create database zabbix character set utf8 collate utf8_bin;
1.6.3创建账户并且授权设置密码
grant all privileges on zabbix.* to zabbix@localhost identified by ‘zabbix’;
#给来自loclhost的用户zabbxi分配可对数据库zabbix所有表进行所有操作的权限，并且设定密码为zabbix
flush privleges #刷新
exit #退出
下面是导入表
cd /usr/share/doc/zabbix-server-mysql-3.2.10/ #进入该目录
gunzip create.sql.gz #解压
登录mysql
mysql -u root -p
切换数据到zabbix
use zabbix
导入sql
source create.sql
退出
exit

#vim /etc/zabbix/zabbix_server.conf
